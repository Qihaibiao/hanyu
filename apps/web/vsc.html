VS Code 插件

在 vscode 左边栏插件市场中搜索下列英文名点击安装即可。

  • Chinese ：中文界面
  • Prettier ：代码美化
  • Wisen-Translate ：翻译插件(云环境下没有)
  • Live preview ：实时预览html(在云环境下改用Live Server)
  • Search/Translate Hero ：搜索跳转和翻译(云环境下没有)
  • vscode-icons ：美化图标
